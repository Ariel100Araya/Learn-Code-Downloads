In HTML, you can make text stand out by adding a few simple tags in the p tag.


To add bold text you add the <p > <b >[Type bold text here] </b > </p >

To add italic text you add the <p > <i >[Type italic text here] </i > </p >

To add underlined text you add the <p > <u >[Type underlined text here] </u > </p >
